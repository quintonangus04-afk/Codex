Fetch Command

The fetch command is used to send & recieve data from a backend server, which is used in services like POS & user authentication systems. There are two main types of fetch requests, GET & POST requests. A GET request is designed to get infomation from the server, while a POST request sends a request to the server to proform a task maybe returning a response. Both can be used in their own cases however POST is the most common used.

POST request

The POST request, as said before, sends a request to a server to proform a function which may return a response. When sending a request to the server it requires four main details:

  1. URL - this tells the frontend where to send the POST request to.
  2. method - tells the frontend what type of request to send, in this case a POST request.
  3. header - which sends infomation about your request like what format it is being sent as, and what format to respond with.
  4. body - this is the data that is sent to the server such as ID, password, and other info.

const request = await fetch('backend server url', {  # Note: await is only required if waiting for a response from the server
    method: 'POST',
    header: '{'Content-Type': 'application/json'}',
    body: {data: data}
})
